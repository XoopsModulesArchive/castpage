Administration Information

Installing

Installation of the castpage module is just like installing any other xoops modle.
Upload the files to your modules directory, and from the Admin Console, select "System Admin" button and select "Modules".
Scroll down towards the bottom and find the cast page icon. Click on "Install" and you are ready to go.

Setup

From the Admin panel, select the cast page icon . Click on "setup".
The options for you to fill out are: 
Title:   The name of the webcomic or site, or whatever.
Next Image:   An image you want displayed if you have multiple chapters.
              This image, when clicked, will goto the next chapter. 
              Image will need to be in the "images" folder under the castpage module.
Prev Image:   An image you want displayed, that when clicked, it will go to
              the previous chapter.
Image size X: These are the size requirements for the prev/next images. 
              Set either X or Y to 0 to remove size limitation.
Image size Y:
First Seen text: Text to be displayed above each castmember to say what chapter they
              where introduced.
Last Seen text:  Text to be dispalyed above each castmember to say when they where last seen.
That's it for the basic setup. Next you need to add chapters and optional sections.

Adding Chapters

One chapter is required. From the cast page icon, click on "Add Chapter".
The options are as follows: 
Chapter Title: The name of the chapter.
Picture Filename: The logo you want on top of the chapter. This logo should
        be placed in the directory listed below.
Directory: the directoy to fine the logo and the images of the cast.
  The direcory should be created inside the "images" directory of the castpage module.

Adding Sections

Sections are optional. These can be used to help sort out the people you have on your cast listing.

From the castpage icon, click on "Add Section". This has only one option, the name of the section.

Adding Cast

Finally, you need to add cast to your pages. Click on "Add a New Cast Member" form the castpage icon. 
Name:   The name of the person
First Seen:  what chapter they are first seen in.  
Last Seen:  what chapter they are last seen in. 
    If these are left blank, they will not be displayed.
Picture Filename:  The image of the person.  Located in the chapter images directory (see "Adding Chapters" above)
Chapter: A dropdown list of the available chapters.
Section: A dropdown list of available sections.
Description: A discription of the cast person.

Viewing Your Cast

Now that you are done entering your information. Simply click on "Cast Page" from the Main Menu and enjoy.
